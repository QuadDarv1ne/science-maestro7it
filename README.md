## Описание

**Science Maestro7IT** — персональный научный портфельный сайт, демонстрирующий 30 научных публикаций автора с платформы Zenodo, 14 проектов с GitHub, а также профессиональную биографию. Реализован на Next.js 16 с TypeScript, Tailwind CSS 4 и shadcn/ui.

Проект является частью образовательной экосистемы **Maestro7IT** — школы программирования, основанной автором.

**Автор:** Дуплей Максим Игоревич — старший преподаватель информационных технологий, аналитик, философ, музыкант и DevOps-инженер.

## Установка и запуск

### Требования

- **Node.js** 18+ или **Bun** 1.0+
- Python 3.10+ (только для пересборки данных из Zenodo/GitHub)

### Установка зависимостей

```bash
# Через Bun (рекомендуется)
bun install

# Или через npm
npm install
```

### Запуск dev-сервера

```bash
bun run dev
# или
npm run dev
```

Сайт будет доступен по адресу `http://localhost:3000`

### Сборка для production

```bash
bun run build
bun run start
```

### Проверка линтером

```bash
bun run lint
```

## Возможности

### Структура страницы

1. **Hero** — имя, ORCID-бейдж, фото автора в декоративной рамке, чипы с ключевыми компетенциями, плавающие бейджи (ORCID, Maestro7IT), 3 CTA-кнопки, быстрая статистика
2. **Об авторе** — 6 ролевых карточек (Преподаватель ИТ, Аналитик, Философ, Музыкант, DevOps-инженер, Основатель Maestro7IT) + цитата с авторскими правами
3. **Направления** — 4 крупных статистических блока + 9 кликабельных карточек категорий с прогресс-барами
4. **Хронология** — гистограмма публикаций по месяцам с анимированным заполнением и hover-подсказками
5. **Публикации** — поиск по тексту, 3 режима сортировки, фильтр-чипы по 9 категориям, сетка карточек с краткой аннотацией, кнопка «Показать ещё», модальное окно с полной аннотацией, шаблоном цитирования и кнопкой «Поделиться»
6. **Проекты** — 14 проектов в 7 категориях с фильтрами и live-счётчиками
7. **Контакты** — карточка профилей (ORCID, GitHub, Stepik, Maestro7IT) + карточка цитирования с кнопкой «Копировать»

### 5 тем оформления

Переключаются через dropdown в шапке или командную палитру:

| Тема | Фон | Акцент |
|------|-----|--------|
| **Светлая** | Кремовый | Янтарный |
| **Тёмная** | Угольный | Янтарный |
| **Фиолетовая** | Индиго | Лавандовый |
| **Оранжевая** | Кофейный | Ярко-оранжевый |
| **Океан** | Тёмно-синий | Бирюзовый |

Выбор сохраняется в localStorage.

### Интерактивность

- **Командная палитра** (`Ctrl/Cmd+K`) — быстрый доступ к навигации, публикациям, проектам, темам и внешним действиям
- **Поиск публикаций** — горячая клавиша `/` для фокуса на поле поиска
- **Анимированный фон** — Canvas созвездие частиц, адаптируется к активной теме, уважает `prefers-reduced-motion`
- **Индикатор прогресса прокрутки** — тонкая полоска вверху страницы
- **Кнопка «Наверх»** — появляется после прокрутки на 80% высоты экрана
- **Поделиться публикацией** — Telegram, VK, X (Twitter), копирование ссылки
- **Print-friendly** — `Ctrl/Cmd+P` даёт чистую библиографию без декора

### Адаптивность

Полная адаптивность от 320px до 4K. Проверено на:
- Мобильный: 390×844 (iPhone 12 Pro)
- Планшет: 768×1024
- Десктоп: 1440×900

## Скрипты генерации данных

Данные о публикациях и проектах загружаются автоматически с внешних API. Чтобы пересобрать данные:

### 1. Загрузка публикаций с Zenodo

```bash
python3 scripts/fetch_zenodo.py
# Сохраняет scripts/zenodo_data.json
```

### 2. Генерация TypeScript-файла публикаций

```bash
python3 scripts/gen_publications_ts.py
# Создаёт src/data/publications.ts с категоризацией
```

### 3. Загрузка проектов с GitHub

```bash
python3 scripts/fetch_github_repos.py
# Сохраняет scripts/github_repos.json (с rate-limit handling)
```

### 4. Генерация TypeScript-файла проектов

```bash
python3 scripts/gen_repos_ts.py
# Создаёт src/data/repos.ts с категоризацией
```
